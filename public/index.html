VidUP Stream Resolver

Turns a TMDB ID into a playable HLS stream from vidup.to. It reads the title page, runs VidUP’s decode logic, probes hosts, and plays the first link that works.

/movie/941109
Finding IDs — search on themoviedb.org. The number in the URL is what you enter (/movie/941109 → 941109, /tv/1396 → 1396). What happens — the resolver requests that path on vidup.to, decodes stream hosts, and plays the first working HLS link. Switching hosts — after Play, pick another server under the player.
